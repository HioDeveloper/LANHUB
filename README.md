# LANHUB
Для работы нужен python 3.12 и выше
Локальный хаб для обмена файлами и общения в одной Wi-Fi сети. A local hub for file sharing and communication on a single Wi-Fi network.
Фишки: IP-чат, предпросмотр картинок, консольное меню.
Инструкция:
Откройте консоль (win + r: cmd)
py -m pip install -r requirements.txt
Распакуйте на рабочем столе (желательно)
    Создатся папка LANHUB, в ней будет hub.py, uploads и другие.
py hub.py или просто откройте его двойным кликом ЛКМ
Выбрать режим (например, StartWA)
